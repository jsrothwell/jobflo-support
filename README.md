# Jobflo — Support & Feedback
> Your personal job search companion for iPhone and Mac — track every application, interview, and gut feeling in one place.

---

## 🐛 Reporting a Bug
Found something broken? I want to know about it.

[Open a new issue](../../issues/new?template=bug_report.md) and include as much of the following as you can:

- **Device** (e.g. iPhone 15, MacBook Pro)
- **Platform** (iOS / macOS / both)
- **OS version** (e.g. iOS 18.3, macOS Sequoia 15.2)
- **What happened** — describe what you saw
- **What you expected** — describe what should have happened
- **Steps to reproduce** — how do I make it happen again?
- **Screenshots or screen recording** — if you have one, attach it

## 💡 Feature Requests
Have an idea that would make Jobflo better? I'd love to hear it.

## 📋 Issue Labels
| Label | Meaning |
|---|---|
| `bug` | Something isn't working correctly |
| `crash` | The app is crashing |
| `ui` | A visual or layout issue |
| `feature-request` | A suggestion for something new |
| `ios` | Specific to the iPhone app |
| `macos` | Specific to the Mac app |
| `wont-fix` | Acknowledged but out of scope |
